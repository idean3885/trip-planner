**shadcn/ui 도입 (base-nova preset, Base UI primitives)**: `src/components/ui/`에 초기 컴포넌트 12종 vendoring — button, input, label, field(form 대체), card, dialog, dropdown-menu, select, tabs, sonner, skeleton, separator. 루트 레이아웃에 `<Toaster />` 배치. 개발 환경 전용 컴포넌트 미리보기 페이지 `/components` 신설(`(dev)` route group + 프로덕션 `notFound()` 이중 차단). shadcn CLI가 자동 주입한 `.dark` 블록은 단일 라이트 테마 정책에 따라 제거(`@custom-variant dark`는 템플릿 컴파일 호환성을 위해 유지, 런타임 무력화). shadcn 신규 의존성: `@base-ui/react`, `class-variance-authority`, `tailwind-merge`, `clsx`, `lucide-react`, `tw-animate-css`, `sonner`.
