# Quickstart: 여행 상세 종일 섹션

## US1 — 종일/시간 분리 렌더

### Evidence

- 자동 테스트: 종일·시간 활동이 섞인 목록에서 종일 활동이 최상단 별도 섹션(기본 접힘)에 모이고, 시간 활동은 그 아래 유지됨을 렌더 검증. 종일 0건이면 섹션 미표시.
- 수동 체크리스트:
  - [ ] 종일·시간 섞인 날 → 최상단 "종일" 섹션 접힌 상태
  - [ ] 펼치면 종일 활동 노출
  - [ ] 종일 0건인 날 → 섹션 없음

## US2 — 종일 지정 입력·표시

### Evidence

- 자동 테스트: 폼에서 종일 토글 시 시각 입력 생략·검증 분기. 카드가 종일 활동을 "종일"로 표시. allDay 저장·재조회 유지.
- 수동 체크리스트:
  - [ ] 활동을 종일로 저장 → 카드 "종일" 표시
  - [ ] 종일 해제 → 시간 활동 복귀

## US3 — 종일 외부 캘린더 반영

### Evidence

- 자동 테스트: 종일 활동 ICS가 `VALUE=DATE` 종일 이벤트로 생성됨. 시간 활동 ICS는 기존 시각 형식 유지(회귀 없음).
- 수동 체크리스트:
  - [ ] 종일 활동 동기화 → 외부 캘린더에 종일 이벤트로 표시
