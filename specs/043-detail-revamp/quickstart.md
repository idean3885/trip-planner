# Quickstart: 여행 상세 화면 정비

## US1 — 여행 기간 직접 편집

### Evidence

- 자동 테스트: 기간 set 동작 단위테스트 — 확장(경계 Day 생성), 축소+활동 보유 시 confirm
  게이트(409 wouldDelete), 손실 없음 즉시 적용, 시작>종료 거부.
- 수동 체크리스트:
  - [ ] 기간 편집 다이얼로그에서 시작/종료 수정
  - [ ] 활동 보유 일자가 범위 밖이 될 때 경고·확인 노출
  - [ ] 손실 없을 때 확인 없이 적용
- 스크린샷: 후속(다이얼로그·경고)

## US2 — 헤더·액션바·캘린더 레이아웃

### Evidence

- 자동 테스트: 정적(타입·lint). 액션바·삭제 버튼 노출 조건은 단위 수준에서 확인 가능분 검증.
- 수동 체크리스트:
  - [ ] 브레드크럼 기간 날짜 표기
  - [ ] 액션 버튼 한 줄 통합
  - [ ] 선택 일자 삭제 노출/비활성 조건
  - [ ] 캘린더 좌우 여백 + 세로 높이 제한
- 스크린샷: 후속(헤더·캘린더)

## US3 — 동기화 진입 단계 축소

### Evidence

- 자동 테스트: 정적. [열기] 제거 후 진입 즉시 섹션 렌더 경로 확인.
- 수동 체크리스트:
  - [ ] 동기화 진입 즉시 내용 노출
  - [ ] 중간 열기 단계 부재
- 스크린샷: 후속

## US4 — 경계 멈춤 스크롤·패널 높이

### Evidence

- 자동 테스트: 정적. 패널 높이 강제 동일화 제거 확인(잔여 min-height/동일 높이 0).
- 수동 체크리스트:
  - [ ] 데스크탑·모바일 경계 1회 멈춤
  - [ ] 짧은 일자에서 빈 스크롤 없음
- 스크린샷: 후속(실기기 검증 필요)
