# Feature Specification: 활동 변경 시 외부 캘린더 자동 반영

## 배경

연결된 외부 캘린더로의 반영이 수동 버튼 전용이다. 활동을 추가·수정·삭제하거나 외부
캘린더에서 일정을 가져와 확정해도 자동으로 반영되지 않는다. 한편 화면 안내는 "추가·
수정 시 자동으로 반영됩니다(별도 조작 없음)"라고 약속해, 안내와 동작이 어긋난다.

## User Scenarios

### US1 — 활동 변경 자동 반영 (P1)

연결된 외부 캘린더가 있는 여행에서 활동을 추가·수정·삭제하거나 외부 일정을 가져와
확정하면, 별도 조작 없이 외부 캘린더에 반영된다.

**Acceptance**
- 활동을 추가하면 연결된 외부 캘린더에 자동 반영된다.
- 활동을 수정·삭제하면 자동 반영된다.
- 외부 일정 가져오기 확정(개별·일괄) 시 자동 반영된다.
- 외부 캘린더가 연결되지 않은 여행에서는 아무 일도 일어나지 않는다.
- 자동 반영이 사용자 응답을 지연시키지 않는다(변경은 즉시 응답).
- 편집 권한이 없는 호출은 반영을 일으키지 않는다.

## Requirements

- **FR-001** 활동 추가·수정·삭제 후 연결된 외부 캘린더에 자동 반영한다.
- **FR-002** 외부 일정 가져오기 확정(개별·일괄) 후 자동 반영한다.
- **FR-003** 외부 캘린더 미연결 여행은 반영 동작을 하지 않는다.
- **FR-004** 자동 반영은 변경 응답을 지연시키지 않도록 응답 이후에 수행한다.
- **FR-005** 편집 권한자만 반영을 일으킨다.
- **FR-006** 자동 반영 실패가 변경 자체(활동 저장)를 되돌리지 않는다.

## Out of Scope

- 외부 캘린더 연결·해제, 가져오기 로직 자체 변경.
- 동기화 충돌 해결 방식 변경(기존 수동 반영 경로의 처리 재사용).

## 용어

- **자동 반영**: 활동 변경 시 사용자 조작 없이 연결된 외부 캘린더에 동기화하는 것.
- **가져오기 확정**: 외부에서 수집한 초안을 여행 활동으로 확정하는 것.
