# Quickstart: 여행 상세 캘린더 재설계

**Feature**: `040-calendar-redesign` | **Spec**: [spec.md](./spec.md) | **Plan**: [plan.md](./plan.md)

## US1 — 가로 grow

### Scenario US1-1: 데스크탑 가로 채움

데스크탑에서 캘린더가 컨테이너 100% 너비, 셀 균등 7등분.

### Evidence

- 자동 테스트: CalendarView/calendar 클래스 정적 검증 (`flex-1`·`w-full`·cell-size 상한 부재).
- 수동 체크리스트:
  - [ ] dev 실기기 데스크탑에서 캘린더 가로 채움 확인
- 스크린샷: docs/evidence/040-calendar/us1-desktop.png (dev 반영 후)
