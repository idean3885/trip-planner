name: 웹/모바일 동일 디자인
description: 반응형 분기 없이 웹/모바일 동일 디자인 유지. 1인 개발 유지보수 우선.
type: feedback
---

웹과 모바일에서 굳이 다른 디자인을 하지 말 것. 동일 디자인으로 통일.

**Why:** 1인 개발이므로 코드와 유지보수 비용을 줄이는 게 우선. 미디어 쿼리로 해상도별 분기하면 관리 포인트가 늘어남.

**How to apply:** 테이블 카드형, 레이아웃 등 UI 컴포넌트를 구현할 때 모바일/데스크톱 분기(미디어 쿼리) 없이 단일 디자인으로 구현. 모바일 우선으로 만들되 데스크톱에서도 그대로 사용.
