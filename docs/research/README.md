## 작성 원칙

- 스펙 본문이나 플랜에 인용한 원자료는 이 디렉토리에 보관하여 후속 의사결정자가 검증 가능하게 둔다.
- 벤더 가격·스펙 등 시점 민감 정보는 파일 상단에 "조사 시점"을 명시한다.
- 결정 결과가 바뀌어도 원자료는 삭제하지 않는다(역사적 맥락 보존).
