# 패킹 리스트

## 의류
- [ ] 여름 옷 **6-7일치만** (2주치 다 싸지 말 것!)
- [ ] 속옷/양말 **속건 소재** (손빨래 후 하룻밤이면 마름)
- [ ] 얇은 긴팔/가디건 (저녁용, 포르투·그라나다 선선)
- [ ] 편한 운동화 (돌길 많음!) + 샌들
- [ ] 모자, 선글라스
- [ ] 수영복 (세비야 루프탑 풀 + 바르셀로나 해변)
- [ ] 격식 있는 옷 1벌 (파인다이닝/플라멩코용)

## 세면/건강
- [ ] 선크림
- [ ] 트래블 세탁세제 소분 (다이소)
- [ ] 상비약

## 여행 용품
- [ ] 압축팩 (더러운 옷 분리 + 공간 절약)
- [ ] 와인 오프너
- [ ] 와인 캐리어/뽁뽁이 (구매한 와인 운반)
- [ ] 여권, 항공권 출력
- [ ] 트레블 월렛 + 현금(유로)

---

## 세탁 계획

| 시점 | 방법 | 비용 |
|------|------|------|
| DAY 1~4 (리스본~포르투) | 호텔에서 속옷/양말 손빨래 | 무료 |
| **DAY 4 포르투** | 빨래방 (Lavandería) 1회 | ~€8 (₩12,000) |
| DAY 5~8 (마드리드~세비야~그라나다) | 손빨래로 버티기 | 무료 |
| **DAY 9 바르셀로나 도착** | 아파트 세탁기로 몰아빨래! | 무료 |
| DAY 12쯤 | 필요 시 한번 더 돌리기 | 무료 |

> 바르셀로나 아파트 예약 시 **세탁기(Washing machine) 유무 꼭 확인!**
